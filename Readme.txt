To use this chatbot:
firstly, go to "https://ollama.com/download" and install according to your system.
If ollama is installed, to check: go to terminal and type ollama. you will get ollama infos if it has been instaled.
Next, Type on terminal: ollama pull neural-chat/

To test this out, simply run on terminal: ollama run neural-chat
Now just use the program to your liking.
